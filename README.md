# NewsG
A basic news app built using Gnews API


# Functionalities 
- Authentication (Local Strategy + Google Oauth + Facebook Oauth)
- Top Headlines
- Particular Keyword (city name or a topic)
- Category-Wise News
- Bookmark a article to read later

# Built With

* HTML
* CSS
* BOOTSTRAP-5
* JAVASCRIPT
* NODEJS
* EXPRESS
* MONGODB
* GNews API
* JQUERY
* FONTAWESOME
* SWEETALERT2
* PASSPORTJS
